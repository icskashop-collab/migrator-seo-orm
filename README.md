# SEO & ORM Dashboard — migrator.me

Дашборд и полный аудит сайта **migrator.me** по состоянию на **04.06.2026**.

## Файлы

| Файл | Что внутри |
|------|------------|
| `index.html` | Интерактивный дашборд (2 вкладки: **SEO-аудит** и **ORM**). Графики на Chart.js, дизайн в стиле migrator.me. Открывается в браузере, ничего ставить не нужно. |
| `SEO-audit-migrator.me-2026-06-04.html` | Полный текстовый отчёт: executive summary, таблица проблем с приоритизацией, детальный разбор по 7 направлениям, подтверждённые цифры, сопоставление источников, план работ. Готов к печати/экспорту в PDF. |

## Источники данных

- Screaming Frog — `internal_all.xlsx` (краулинг 31 URL)
- xtool.ru — `migrator_me_seo_data.json`, PDF-отчёт, готовый дашборд
- Бэкап сайта (HTTrack, 38 HTML)
- Живой сайт — robots.txt, sitemap.xml (180 URL), выборочные проверки

Сторонние метрики (XT, TF/CF, оценки трафика) — индикативные.
Пробелы данных: полевые Core Web Vitals и статистика Google Search Console (требуют отдельного съёма).

## Как смотреть

Открыть `index.html` в любом браузере. Для GitHub Pages — он уже назван `index.html`, раздел Pages подхватит его автоматически.
